# Paul Graham Holo Personality

An interactive digital personality based on Paul Graham's essays and writings, accessible at [pg.getcore.me](https://pg.getcore.me).

## Overview

This holo is built using Paul Graham's essays as input. [CORE](https://github.com/redplanethq/core) has generated a collection of common documents that you can explore and chat with at [pg.getcore.me](https://pg.getcore.me).

## Topics Covered

The personality includes knowledge on:
- **Startups** - Building companies, startup advice, and entrepreneurship
- **Life** - Essays on life, work, and philosophy
- Technology and programming
- Writing and communication

## Data Source

The data for this personality comes from publicly available essays written by Paul Graham on [paulgraham.com](https://paulgraham.com).

## Files

- `holo.json` - HOLO configuration with navigation and CORE integration
- `introduction.mdx` - Welcome page introducing the personality
- `pg.webp` - Profile image
- `Dockerfile` - Deployment configuration
- `.env` - Environment configuration (not tracked in git)

## Usage

Visit [pg.getcore.me](https://pg.getcore.me) to:
- Ask questions about startups and entrepreneurship
- Explore Paul Graham's thoughts on technology and writing
- Chat with the digital personality based on his essays

## Disclaimer

⚠️ **Important**: This digital personality is generated from publicly available essays and can make mistakes. It is not an official representation and should not be considered as authoritative advice. Always verify important information from original sources.
