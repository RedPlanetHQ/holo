# Y Combinator Holo Personality

An interactive digital personality based on Y Combinator content, accessible at [yc.getcore.me](https://yc.getcore.me).

## Overview

This holo is built using content from Y Combinator's YouTube videos from the last year. [CORE](https://github.com/redplanethq/core) has generated a collection of documents from this content that you can explore and chat with at [yc.getcore.me](https://yc.getcore.me).

## Topics Covered

The personality includes knowledge on:
- Startup funding and the YC program
- Accelerators and how they work
- Building and scaling companies
- Advice from YC partners and founders
- The startup ecosystem

## Data Source

The data for this personality comes from publicly available YouTube videos from Y Combinator's official channel, covering the last year of content.

## Files

- `holo.json` - HOLO configuration with navigation and CORE integration
- `introduction.mdx` - Welcome page introducing the personality
- `yc.png` - Profile image
- `Dockerfile` - Deployment configuration
- `.env` - Environment configuration (not tracked in git)

## Usage

Visit [yc.getcore.me](https://yc.getcore.me) to:
- Ask questions about startup funding and the YC program
- Learn about building successful companies
- Explore insights from YC partners and founders
- Get advice on accelerators and startup programs

## Disclaimer

⚠️ **Important**: This digital personality is generated from publicly available YouTube content and can make mistakes. It is not an official Y Combinator resource and should not be considered as authoritative advice. Always verify important information from official YC sources.
